📘 13. Inner Classes

Java allows you to define a class within another class. Such classes are called Inner Classes. They can access members (including private) of the outer class and help logically group classes that are only used in one place.

🔹 1. Member Inner Class

A non-static class defined inside another class. It has access to all members of the outer class.

class Outer {
    private String msg = "Hello from Outer";

    class Inner {
        void display() {
            System.out.println(msg);
        }
    }

    public static void main(String[] args) {
        Outer outer = new Outer();
        Outer.Inner inner = outer.new Inner();
        inner.display();
    }
}

🔹 2. Static Nested Class

A static class defined inside another class. It can only access static members of the outer class directly.

class Outer {
    static int data = 50;

    static class Nested {
        void msg() {
            System.out.println("Static data: " + data);
        }
    }

    public static void main(String[] args) {
        Outer.Nested obj = new Outer.Nested();
        obj.msg();
    }
}

🔹 3. Anonymous Inner Class

A class without a name used for instantiating classes or interfaces. Often used for event handling or thread creation.

abstract class Animal {
    abstract void sound();
}

public class Test {
    public static void main(String[] args) {
        Animal a = new Animal() {
            void sound() {
                System.out.println("Roar");
            }
        };
        a.sound();
    }
}

🔹 4. Local Inner Class

Defined inside a method and can access local variables (must be final or effectively final).

class Outer {
    void outerMethod() {
        int num = 100;

        class LocalInner {
            void print() {
                System.out.println("Value: " + num);
            }
        }

        LocalInner inner = new LocalInner();
        inner.print();
    }

    public static void main(String[] args) {
        Outer o = new Outer();
        o.outerMethod();
    }
}
