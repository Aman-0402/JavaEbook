📗 2. Java Basics

🔷 Writing Your First Java Program (HelloWorld.java)

// HelloWorld.java
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
  • public class HelloWorld: Defines a class named HelloWorld.
  • public static void main(String[] args): Entry point of any Java program.
  • System.out.println(): Prints output to the console.

🔷 Java Syntax & Structure

  • Every Java application must have at least one class.
  • Statements end with a semicolon ;.
  • Java is case-sensitive.
  • Curly braces {} define code blocks.
  • The main method is the program’s entry point.
public class ClassName {
    public static void main(String[] args) {
        // Your code here
    }
}

🔷 Type Casting

Implicit casting (Widening): Automatic conversion from smaller to larger types.

int i = 100;
long l = i;  // int to long

Explicit casting (Narrowing): Manual conversion from larger to smaller types.

double d = 100.04;
int i = (int) d;  // double to int, fractional part lost

🔷 Constants and Literals

Literals are fixed values written directly in code. Use the final keyword to declare constants (values that don’t change).

final double PI = 3.14159;

🔷 Comments in Java

  • Single-line comment: // This is a comment
  • Multi-line comment:
    /* This is
    a multi-line
    comment */
  • Documentation comment:
    /** This is a documentation comment */
