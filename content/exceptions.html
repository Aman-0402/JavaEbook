üî∑ What is an Exception?

An exception is an event that disrupts the normal flow of the program. It is an object that is thrown at runtime when an error occurs.

üî∑ Types of Exceptions

TypeDescription
Checked ExceptionsChecked at compile time (e.g., IOException, SQLException)
Unchecked ExceptionsChecked at runtime (e.g., ArithmeticException, NullPointerException)
ErrorSerious problems like JVM crash, out of memory (e.g., StackOverflowError)

üî∑ try-catch block

The try block contains code that might throw an exception. The catch block handles the exception.

public class TryCatchExample {
    public static void main(String[] args) {
        try {
            int result = 10 / 0;
        } catch (ArithmeticException e) {
            System.out.println("Exception caught: " + e);
        }
    }
}

üî∑ finally block

The finally block always executes whether an exception is caught or not. It is used to release resources like files or database connections.

try {
    int[] arr = {1, 2};
    System.out.println(arr[5]);
} catch (ArrayIndexOutOfBoundsException e) {
    System.out.println("Array error: " + e);
} finally {
    System.out.println("This block always executes.");
}

üî∑ throw and throws

  • throw is used to explicitly throw an exception.
  • throws is used to declare exceptions a method might throw.
public class ThrowExample {
    static void validate(int age) {
        if(age < 18)
            throw new ArithmeticException("Not eligible to vote");
        else
            System.out.println("Welcome to vote");
    }
    public static void main(String[] args) {
        validate(16);
    }
}
import java.io.*;

public class ThrowsExample {
    void readFile() throws IOException {
        FileReader fr = new FileReader("file.txt");
        BufferedReader br = new BufferedReader(fr);
    }
}

üî∑ Custom Exceptions

You can define your own exception classes by extending the Exception class.

class MyException extends Exception {
    MyException(String message) {
        super(message);
    }
}

public class CustomEx {
    public static void main(String[] args) {
        try {
            throw new MyException("This is a custom exception!");
        } catch (MyException e) {
            System.out.println("Caught: " + e.getMessage());
        }
    }
}
