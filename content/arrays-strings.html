📘 5. Arrays and Strings

🔷 One-dimensional Arrays

An array is a container object that holds a fixed number of values of a single type.

int[] numbers = new int[5];
numbers[0] = 10;
numbers[1] = 20;
System.out.println(numbers[0]);

🔷 Two-dimensional Arrays

A 2D array is an array of arrays. Useful for matrices or tables.

int[][] matrix = {
    {1, 2, 3},
    {4, 5, 6}
};
System.out.println(matrix[1][2]); // Output: 6

🔷 Array Methods

  • Arrays.sort(arr) – Sorts the array
  • Arrays.toString(arr) – Returns a string representation
  • Arrays.copyOf(arr, newLength) – Copies the array
import java.util.Arrays;
int[] a = {5, 3, 1};
Arrays.sort(a);
System.out.println(Arrays.toString(a));

🔷 Enhanced for-loop

Used to iterate through arrays or collections easily.

int[] nums = {1, 2, 3};
for (int num : nums) {
    System.out.println(num);
}

🔷 String Basics

Strings are objects in Java that represent sequences of characters.

String name = "Java";
System.out.println(name.length());

🔷 Common String Methods

MethodDescription
length()Returns length of string
charAt(index)Returns character at index
substring(begin, end)Returns part of the string
equals(str)Checks equality
toUpperCase()Converts to uppercase
indexOf(char)Finds index of character

🔷 StringBuilder vs StringBuffer

AspectStringBuilderStringBuffer
Thread-SafeNoYes ✅
PerformanceFasterSlower
MutabilityMutableMutable
StringBuilder sb = new StringBuilder("Hello");
sb.append(" Java");
System.out.println(sb.toString());
