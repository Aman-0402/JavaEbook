üìò 14. Java 8 Features (Intro)

Java 8 introduced several major features aimed at improving the efficiency, readability, and conciseness of Java code. These include lambda expressions, the Stream API, method references, and functional interfaces.

üîπ Lambda Expressions

Lambda expressions allow us to write anonymous methods (functions) in a concise way.

Syntax: (parameters) -> expression or block

interface Greeting {
    void sayHello();
}

public class LambdaDemo {
    public static void main(String[] args) {
        Greeting greet = () -> System.out.println("Hello, Java 8!");
        greet.sayHello();
    }
}
  • Can be used to implement interfaces with a single abstract method (functional interfaces).
  • Improves readability and reduces boilerplate code.

üîπ Functional Interfaces

An interface with exactly one abstract method is called a functional interface. Java 8 provides a new annotation @FunctionalInterface.

@FunctionalInterface
interface MyFunction {
    void execute();
}

public class Test {
    public static void main(String[] args) {
        MyFunction f = () -> System.out.println("Functional Interface in action!");
        f.execute();
    }
}

Common Functional Interfaces in java.util.function package:

InterfaceDescription
Predicate<T>Returns boolean value
Function<T, R>Returns value of type R
Consumer<T>Performs operation on T
Supplier<T>Returns value of type T

üîπ Method References

Method references provide a way to refer to methods without executing them. It uses :: operator.

import java.util.function.Consumer;

public class MethodRef {
    static void sayHi(String name) {
        System.out.println("Hi " + name);
    }

    public static void main(String[] args) {
        Consumer<String> con = MethodRef::sayHi;
        con.accept("Aman");
    }
}

Types of Method References:

  • ClassName::staticMethod
  • object::instanceMethod
  • ClassName::new (Constructor reference)

üîπ Stream API Basics

The Stream API provides functional-style operations on sequences of elements (like a list or array).

import java.util.Arrays;
import java.util.List;

public class StreamExample {
    public static void main(String[] args) {
        List<Integer> numbers = Arrays.asList(1, 2, 3, 4, 5);
        numbers.stream()
               .filter(n -> n % 2 == 0)
               .map(n -> n * n)
               .forEach(System.out::println);
    }
}

Common Stream Operations:

  • filter(): Filters elements based on a condition
  • map(): Transforms elements
  • forEach(): Performs an action for each element
  • collect(): Converts stream to list, set, etc.
